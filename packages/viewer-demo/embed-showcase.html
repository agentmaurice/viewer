Viewer Embed Demo

Three OpenUI mini-apps embedded in a static HTML page

This page shows how to use @agent-maurice/viewer-embed without a React application. The component is imported directly in the <head>, then several <agent-maurice-viewer> elements are placed in the HTML with a mock-scenario.

Import JS
<script type="module"> import '@agent-maurice/viewer-embed' </script> <agent-maurice-viewer mock-scenario="sales-cockpit" theme-primary="#1d4ed8" theme-radius="18px" ></agent-maurice-viewer>
0 React components to write
3 mocked mini-apps on the page
1 JS import in the header
What it shows
  • Several embeds side by side in a simple web page.
  • A local stub to test the viewer without a backend or Supabase.
  • Different themes per instance through HTML attributes.
  • A simple base that can later replace the mock with a real slug or deployment.
sales-cockpit KPI Table

Sales cockpit

Control-oriented example with segments, focus areas, and a deals table.

ops-review Review Form

Operations review

More workspace-oriented example with work queues, priorities, and demo annotation.

support-center Monitoring Multi-embed

Support control center

Monitoring example with alerts and a more critical tone, useful for demonstrating a supervision embed in a static page.
