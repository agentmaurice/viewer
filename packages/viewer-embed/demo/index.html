AgentMaurice Viewer — Embed Demo

Discover agent-maurice-viewer, an embeddable Web Component for integrating AgentMaurice into any static site. Each example below shows a different usage with theme customizations.

Contact form — Default theme

This example renders a contact form with AgentMaurice's default color palette (primary violet #7C3AED).

<agent-maurice-viewer
deployment-id="demo_deploy"
recipe-id="contact_form"
api-url="http://localhost:8080"
theme-primary="#7C3AED"
theme-radius="8px"
></agent-maurice-viewer>
Deployment Form Contact

Feedback form — Custom theme

This example uses a green theme (#059669) with rounder corners (16px) for a more modern look.

<agent-maurice-viewer
deployment-id="demo_deploy"
recipe-id="feedback_form"
api-url="http://localhost:8080"
theme-primary="#059669"
theme-radius="16px"
></agent-maurice-viewer>
Deployment Feedback Custom theme

Form with initial state

This form is initialized with seed data (initial state). Fields can be prefilled through the seed-state attribute.

<agent-maurice-viewer
deployment-id="demo_deploy"
recipe-id="profile_form"
api-url="http://localhost:8080"
theme-primary="#0891B2"
seed-state='{...}'
></agent-maurice-viewer>

Available attributes

Here is the full list of attributes you can use to configure an <agent-maurice-viewer>.

Attribute Required Description
deployment-id Yes ID of the AgentMaurice deployment to load
recipe-id Yes ID of the recipe (meta-recette) to execute
api-url No AgentMaurice API URL (default: https://api.agentmaurice.dev)
theme-primary No Primary color (for example: #7C3AED)
theme-secondary No Secondary color (for example: #0891B2)
theme-radius No Border radius (for example: 8px, 16px)
theme-font-family No Custom font family
seed-state No Initial JSON state (for example: '{\"key\":\"value\"}')
